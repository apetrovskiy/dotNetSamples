

In folder Tests/TutorialTests/.Dummies is a file .Aliases.cs.

It contains the dummy type used in the tests & #define USE_DUMMIES.
Uncomment #define USE_DUMMIES for using dummy types (replacing the incomplete types of JDI-Framework).
It is necessary to fully build the project.

Сomment #define USE_DUMMIES for using complete types of JDI-Framework.
The project can not be built.

To use the type from JDI-Framework you need to comment #define USE_DUMMIES.
